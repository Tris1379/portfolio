# Phase Quick Task 260324-b49: Cinematic Maple Leaf Storm Page Transition Summary

Cinematic maple leaf storm burst with depth-of-field that floods the screen during page navigation.

## One-Liner

Added a dual-layer tsParticles burst (80-120 small sharp + 8-15 large CSS-blurred leaves) that activates on every page navigation via `isTransitioning` state in PageContext, with coordinated timing: content exits at 0.35s, page switches at 0.4s peak, new content fades in at 0.45s, storm disperses by 0.9s.

## What Was Built

### 1. PageContext Transition State (`src/components/PageContext.tsx`)
- Added `isTransitioning: boolean` to `PageContextValue` interface
- `navigateTo` sets `isTransitioning = true` immediately, then schedules:
  - `setCurrentPage(page)` at 400ms (peak leaf density)
  - `isTransitioning = false` at 900ms (storm complete)
- `transitionTimerRef` + `postSwitchTimerRef` clear on rapid navigation (prevents overlapping storms)
- Both timers cleaned up on unmount

### 2. MapleLeafStorm Component (`src/components/MapleLeafStorm.tsx`)
- Lazy-loads `@tsparticles/react` + `@tsparticles/slim` (shares global engine with ParticlesBackground)
- Only mounts when `isTransitioning === true` â€” unmounting disposes all particles instantly
- **Two particle layers:**
  - Small leaves: 100 particles, 10-30px, sharp, speeds 3-8
  - Large leaves: 12 particles, 35-50px, wrapped in `filter: blur(2.5px)`, speeds 5-12
- Both use leaf1.png/leaf2.png assets with tilt, rotate, and life duration (0.6-1.0s)
- Responsive: 40 small + 5 large on mobile (<768px)
- Fixed overlay, `pointer-events-none`, z-index via `var(--z-transition-particles)`

### 3. Page Integration (`src/app/[locale]/page.tsx`)
- `<MapleLeafStorm />` renders inside `<main>` but outside `<AnimatePresence>`
- Updated `pageVariants` timing:
  - **Exit:** 0.35s (faster, content gone before leaf peak)
  - **Animate:** 0.5s duration with 0.45s delay (fades in as leaves disperse)

### 4. CSS Token (`src/app/globals.css`)
- Added `--z-transition-particles: 40` (above content/cards, below navbar at 50)
